# vis-lut

> **WIP!**

A [vis](https://github.com/martanne/vis) theme inspired by Vim's builtin desert theme, heavily modified to my liking :).

## Usage

Just do:

~~~bash
% wget
~~~

And add this to your `visrc.lua`:

~~~lua
vis:command("set theme lut")
~~~
